# Voice-Phishing-Detection-System

## Overview
This project detects voice phishing (vishing) attacks using machine learning techniques.

## Features
- MFCC feature extraction
- Audio preprocessing
- Classification using ML models
- Real-time detection capability

## Technologies Used
- Python
- Machine Learning
- Signal Processing

## How It Works
1. Audio input is collected
2. MFCC features are extracted
3. Model classifies as phishing or legitimate

- Improve accuracy
